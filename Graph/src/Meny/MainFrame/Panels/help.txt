Help для работы с программой
    Справка по компонентам редактора графа

            Поле <b>Redact Area</b> - Зона, в которую необходимо ввесли список инцидентности
            Кнопка <b>Build Graph</b> - Строит и рисует граф по списку инцидентности
            Поле <b>Insert:Source node name</b> - Имя вершины, которую необходимо добавить в граф
            Поле <b>Isert:List of neighbors nodes</b> - Список смежных вершин для введённой вершины
            Кнопка <b>Add node</b> - добавляет вершину в граф
            Кнопка <b>Delete node</b> - удаляет выбранную вершину
            Поля <b>Source node name</b> и <b>Dest node name</b> - список всех вершин в графе
            Поле <b>List of neighbors nodes</b> - Список смежных вершин для введённой Source Node вершины
            Кнопка <b>Add Connection</b> - добавляет ребро между Source Node и Dest Node
            Кнопка <b>Delete Connection</b> - удаляет ребро между Source Node и вершиой из List of neighbors nodes
        Справка о компонентах управления алгоритмом

            <li>Кнопка <b>Start algorithm</b> - Запускает алгоритм нахождения мостов и визуализирует результат<br>
            Если Step by step solution активен, то будет показана пошаговая работа алгоритма<br>
            Если Step by step solution не активен, то будет показан только конечный результат</li>
            <li>Кнопка <b>Start manually</b> - Позволяет выполнить алгоритм по нахождению мостов по шагам<br>
            После нажатия на кнопку, будет выполнен один шаг алгоритма, после чего программа нарисует текущее состояние графа</li>
