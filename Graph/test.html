Help для работы с программой

Справка по компонентам редактора графа

  1. Поле Redact Area - Зона, в которую необходимо ввесли список инцидентности
  2. Кнопка Build Graph - Строит и рисует граф по списку инцидентности
  3. Поле Insert:Source node name - Имя вершины, которую необходимо добавить в граф
  4. Поле Isert:List of neighbors nodes - Список смежных вершин для введённой вершины
  5. Кнопка Add node - добавляет вершину в граф
  6. Кнопка Delete node - удаляет выбранную вершину
  7. Поля Source node name и Dest node name - список всех вершин в графе
  8. Поле List of neighbors nodes - Список смежных вершин для введённой Source Node вершины
  9. Кнопка Add Connection - добавляет ребро между Source Node и Dest Node
  10. Кнопка Delete Connection - удаляет ребро между Source Node и вершиой из List of neighbors nodes

Справка о компонентах управления алгоритмом

  1. Кнопка Start algorithm - Запускает алгоритм нахождения мостов и визуализирует результат
    Если Step by step solution активен, то будет показана пошаговая работа алгоритма
    Если Step by step solution не активен, то будет показан только конечный результат
  2. Кнопка Start manually - Позволяет выполнить алгоритм по нахождению мостов по шагам
    После нажатия на кнопку, будет выполнен один шаг алгоритма, после чего программа нарисует текущее состояние графа
